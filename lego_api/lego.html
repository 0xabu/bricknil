Remote buttons¶

Two buttons on port 0 and 1

  • Port 0 - Left side
  • Port 1 - Right side
Modes and returned values:
# rckey
  • 0 = Nothing pressed
  • 1 = ‘+’ button pushed
  • 255 = ‘-‘ button pushed
  • 127 = ‘red’ button pushed

# keyr same # keya same

# keyd -
  • 0 = Nothing pressed
  • 1 = ‘+’
  • 2 = red
  • 4 = ‘-‘
# keysd bitmask (can tell when multiple buttons are being pressed)
  • [0,0,0] = Nothing pressed
  • [1,0,0] = ‘+’
  • [0,1,0] = red
  • [0,0,1] = ‘-‘
